# STM32_F411_Study
STM32 NUCLEO-F411RE 嵌入式学习仓库

## 简介
广西民族大学自动化专业学生，基于 STM32CubeMX + HAL 库的学习记录。
内容包括基础实验、驱动代码、综合项目，持续更新。

## 学习内容
- GPIO / 按键 / LED 控制
- UART 串口通信
- I2C 驱动 OLED、DHT11 温湿度传感器
- 定时器、PWM、ADC 实验
- 综合项目：环境监测与报警系统

## 开发环境
- STM32CubeMX
- Keil MDK
- NUCLEO-F411RE

## 项目规划
1. 基础外设实验
2. 传感器与显示模块
3. 综合课程设计
4. 就业向作品整理
