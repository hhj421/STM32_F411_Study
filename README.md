# STM32_F411_Study
STM32 NUCLEO-F411RE 嵌入式学习与项目记录

## 个人简介
广西民族大学自动化专业，专注嵌入式软件开发学习，
注重实践与工程规范，项目与代码持续更新。

## 技术栈
- 主控：STM32F411RE（Cortex-M4）
- 开发工具：STM32CubeMX + Keil MDK
- 语言：C 语言
- 通信协议：GPIO / UART / I2C / PWM

## 学习内容
- LED 流水灯、按键输入、外部中断
- 串口打印、数据上传调试
- OLED 显示屏驱动
- DHT11 温湿度传感器采集
- 定时器、PWM、ADC 基础实验
- 综合项目：环境监测与智能报警系统

## 项目规划
1. 基础外设实验
2. 传感器与显示模块驱动
3. 综合课程设计
4. 就业向项目整理与优化

## 目标
具备独立完成嵌入式项目开发、调试与问题排查能力，
能够胜任嵌入式助理工程师岗位。
